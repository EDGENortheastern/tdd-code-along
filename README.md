# TDD code along

To run the project locally:

With SSH:

```bash
git clone git@github.com:EDGENortheastern/tdd-code-along.git
```

Without SSH:

```bash
git clone https://github.com/EDGENortheastern/tdd-code-along.git
```

To install Jest:

```bash
npm install
```

To run tests:

```bash
npm test
```
